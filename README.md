# Mutation Arena 🎮

Mutation Arena is a fast-paced experimental arena game developed using the Unity Game Engine. The game places players inside a dynamic combat arena where mutations, abilities, and survival strategies determine who dominates the battlefield.

This project was created as a learning and development project to explore gameplay mechanics, character abilities, and arena-based combat systems using Unity.


## 🚀 Features

* Real-time arena combat
* Player mutation system
* Ability-based gameplay
* Enemy AI interactions
* Interactive environment
* Modular game architecture
* Unity physics and animation integration



## 🛠️ Built With

* Unity Game Engine
* C#
* Unity Physics System
* Unity Animation System
* Git for Version Control


## ⚙️ Installation

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/ThanuHith/MutationArenaUnity.git
```

### 2. Open in Unity

1. Open Unity Hub
2. Click **Add Project**
3. Select the cloned project folder
4. Open the project with the recommended Unity version

### 3. Run the Game

Open the main scene from:

Assets → Scenes

Then press **Play** in the Unity Editor.

Players must:

* Fight enemies
* Survive longer rounds
